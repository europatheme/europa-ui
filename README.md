# How to install?

### Flutter

```bash
flutter pub add europaui_components
```
But, if you prefer, you can install and configure manually.

### React

#### Using Yarn:

```bash
yarn add @europaui/lib
```

#### Using NPM:

```bash
npm i @europaui/lib
```
